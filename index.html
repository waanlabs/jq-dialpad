Welcome To jq-dialpad

The jQuery dialpad or simply knows as the jq-dialpad is a light plugin that is developed using HTML5 and CSS3. The underlying core is the jQuery library. This plugin can be used in various projects that would need a dialpad from the simple widget application to a fully fledge application.


The plugin is being developed with bootstrap to make sure that the each element in the Dom will adopt itself to various devices enabling the user to have a smooth and user-friendly experience. The jq-dialpad is easy to use and it’s a multifunctional jQuery plugin 

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Usage

The plugin contains all source files. Once you download the source file extract it to a desired location. Make sure the following files and folders are available

-css
     |- bootstrap
     |- font-awesome.min
     |- font-awesome-ie7.min
     |- ie
     |- main
     |- normalize
-fonts
     |- FontAwesome
     |- fontawesome-webfont
     |- fontawesome-webfont
     |- fontawesome-webfont
     |- fontawesome-webfont
     |- glyphicons-halflings-regular
     |- glyphicons-halflings-regular
     |- glyphicons-halflings-regular
     |- glyphicons-halflings-regular
-images
     |- calc-icon-large
-js
     |- bootstrap.min
     |- jquery-1.10.2.min
     |- main
     |- main-mini
     |- modernizr-2.6.2.min
     |- plugins
-saas
     |- bootstrap
     |- ie
     |- main
     |- normalize

The source code is self-explanatory so if you which to use jq-dialpad just include the html that is available and included all dependencies (js,css)
